# User & Project Management Portel
this is a simple React Admin Panel built with React, React Router, React-Bootstrap
It includes clean pages for managing Users and Projects, protected routes, and a responsive layout.
## Features
=> Authentication context with protected routes  
=> Dashboard page with navigation 
=> Add / Edit / Delete functionality for users and projects
=> Fully styled using React-Bootstrap
=> Clean folder structure

# Tech stack
React(Frontend Library)  
React Router DOM (Routing)  
React-Bootstrap (UI components)  
Bootstrap (CSS styling)
